‚è±Ô∏è PREPARATION

Technician places Robot Athlete at starting point, before time control sensor. Robot Athlete must be in standby mode, ready to start as soon as start button pressed or button on remote control activated.

üèÅ START ‚Äì RACE PROCEDURE

  1. Technician starts Robot Athlete after referee's whistle by pressing Start button on Robot Athlete or through controller
  2. Timer begins when Robot Athlete passes in front of time control sensor
  3. Robot Athlete must start within next 5 seconds
  4. If doesn't start within that time, referee allows only one restart (only for first attempt)
  5. Robot Athlete must begin moving quickly, following black line
  6. Goal: finish course in shortest possible time
  7. If exceeds 3 minutes, referee ends round with whistle. Time recorded as 180 seconds (0 points)
  8. Robot Athlete (or any part) not allowed to exit boundaries. If happens = attempt failed
  9. Robot allowed to leave track for max 3 seconds during run (e.g., turns/zigzagging while following line)
  10. Robot must continue following line in correct direction. If completely exits track or loses direction = attempt invalid
  11. If track larger than described dimensions, expected time increased
  12. If Robot Athlete fails to start, time recorded as 180 seconds (0)

üîÅ ROUNDS ‚Äì ATTEMPTS - COMPETITION PROCEDURE

  • Duration: 90 minutes (may be extended depending on participants)
  • Each Robot Athlete makes repeated attempts according to order announced by organizing committee
  • If Technician not present when their turn = loses attempt, next Technician proceeds
  • Missed Technician must wait until all other attempts completed for turn to come again
  • Before each attempt: secondary technical inspection by assistant referee
  • Referee records time of Robot Athletes for each attempt
  • Team responsibility to ensure each attempt within allocated time window
  • Between attempts: corrections to Robot Athletes and code/algorithm allowed, but no testing

‚ùå END OF A ROUND

Round ends in following cases:

  1. When Robot Athlete passes in front of time control sensor
  2. If Robot Athlete exits boundaries of arena
  3. If Robot Athlete loses the line
  4. If Robot Athlete becomes immobilized for any reason
  5. If Robot Athlete exceeds 3-minute time limit

  • Robot Athlete doesn't comply with specifications and team refuses required adjustments
  • Technicians behave inappropriately, use offensive language, provoke, or attack participants/referees
  • Discovered Robot Athlete not operating autonomously but remotely controlled (Bluetooth, Wi-Fi, etc.)
