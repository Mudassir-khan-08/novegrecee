üéØ GOAL

Goal of Robot Bowling is for robot to knock down pins using ball, operating autonomously.

Winning team = one that knocks down most pins with fewest attempts.

  • Team Size: 2-3 members

üèÜ COMPETITION

‚è±Ô∏è PREPARATION

  • Robot Athlete placed in Start Zone (20√ó20cm square) facing towards lane (towards pins, not ball)
  • Ball located in Ball Retrieval Area (20√ó8cm) at side of Start Zone
  • Robot Athlete must autonomously move to collect ball before attempting throw
  • Technician takes position behind robot, may only press Start button when instructed by referee

üèÅ START - PROCESS OF THE RACE

  1. Technician places Robot Athlete in Start Zone, waits for referee's signal
  2. Upon referee's instruction, Technician presses Start button
  3. After pressing Start button, Technician must immediately step back for safety
  4. From that moment, Robot Athlete must operate fully autonomously
  5. Robot Athlete moves from Start Zone, retrieves ball from Ball Retrieval Area, proceeds to release it towards pins
  6. Ball must roll only along bowling lane (3.65m). If ball enters side gutters = no pins counted
  7. Robot Athlete has up to 5 seconds to begin moving after activation
  8. Each frame/attempt consists of up to 2 throws (as in standard bowling rules)
  9. If ball doesn't reach pins or leaves lane sideways = throw invalid (0 pins)
  10. If Robot Athlete fails to release ball = attempt invalid
  11. Only one restart permitted during first attempt (1st frame)
  12. Throw must be executed from behind foul line. If any robot part crosses foul line before releasing ball = throw invalid, scored 0 pins

üîÅ ROUNDS ‚Äì ATTEMPTS

  • Robot Bowling match played over 5 Frames
  • Teams compete alternately by Frame: All teams complete 1st Frame before 2nd Frame begins
  • Playing order determined by referee before start, remains same throughout all Frames

In Each Frame:

  • Robot Athlete has up to 2 rolls to knock down 10 pins
  • If Strike achieved = Frame ends immediately, robot waits for next Frame
  • If no Strike achieved = second roll performed for remaining pins

After end of each Frame, pins reset by referee to initial triangular formation (1‚Äì2‚Äì3‚Äì4). Match continues until all 5 Frames completed by all teams.

üé≥ SCORING PER ROLL

Strike (30 points)

If Robot Athlete knocks down all 10 pins on first roll:

  • Strike recorded
  • Frame ends immediately (no second roll performed)
  • Pins reset, game continues with next Frame

Spare (20 points)

If Robot Athlete knocks down all 10 pins in total with two rolls:

  • Spare recorded
  • Frame ends

Open Frame (points per pin)

If pins remain standing after two rolls:

  • Score = 1 point per pin knocked down
  • Example: if 7 pins fall in total ‚Üí score = 7

‚ùå END OF AN ATTEMPT

Attempt ends in following cases:

  1. When Robot Athlete completed its roll(s) for current Frame (either 1 roll for Strike, or 2 rolls otherwise)
  2. If Robot Athlete becomes immobilized and cannot continue
  3. If Robot Athlete throws ball outside lane or into side gutters
  4. If Robot Athlete fails to release ball
  5. If Robot Athlete doesn't start moving within 5 seconds after activation
  6. If ball gets stuck and doesn't move further = throw considered finished, fallen pins counted
  7. If Robot Athlete touched/assisted by team during attempt (except when allowed by referee for safety) = attempt immediately terminated, scored 0 pins

üü• TEAM DISQUALIFICATION

Team disqualified and must withdraw (results not counted in official rankings) if:

  • Robot Athlete doesn't comply with specifications and team refuses required adjustments
  • Technicians behave inappropriately, use offensive language, provoke, or attack participants/referees
  • Discovered Robot Athlete not operating autonomously but remotely controlled (Bluetooth, Wi-Fi, etc.)

‚úÖ ALLOWED / ‚ùå PROHIBITED

‚úÖ ALLOWED:

  • Extension of Robot Athlete after match start by up to 10cm from only one side (any side)
  • Use of any kind of rubber bands for launching/throwing ball

‚ùå PROHIBITED:

  • Robot Athletes using parts that can harm spectators
  • Use of adhesives to improve adhesion
  • Use of pneumatic devices
  • Breaking Robot Athlete into pieces during race
  • Remote control
  • Wireless connection (Bluetooth/Wi-Fi) with computer or any electronic device during match. Teams found with wireless connection (members or coaches) during match = disqualified
  • Restrictions listed in Athlete Robot technical control section
